## Network Access Layer

- This layer is concerned with how devices deliver data to other devices on a network.
- We can say it encompasses the function of the three lower layers of the OSI model.

## Internet Layer

- The Internet Protocol (IP) is the most important protocols in this layer.
- Provides packet delivery on which networks are built. All data in TCP/IP goes throw IP.

## Transport Layer

- [User Datagram Protocol](udp)
- [Transmission Control Protocol](tcp)

## Application Layer

- Telnet
- FTP
- SMTP
- HTTP
