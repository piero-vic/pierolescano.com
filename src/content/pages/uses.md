## Software

<!-- TODO: Insert a good screenshot of my environment -->

### Environment

- [Sway](https://github.com/swaywm/sway) - Wayland compositor
- [Waybar](https://github.com/Alexays/Waybar) - Status bar
- [mako](https://github.com/emersion/mako) - Notification daemon
- [Rofi](https://github.com/davatorium/rofi) - Application and script launcher

### CLI and TUI

- [aura](https://github.com/fosskers/aura) - Package manager
- [zsh](https://www.zsh.org/) - Shell
- [neovim](https://github.com/neovim/neovim) - Text editor
- [lf](https://github.com/gokcehan/lf) - Terminal file manater
- [tmux](https://github.com/tmux/tmux) - Terminal multiplexer

### GUI

- [Alacritty](https://github.com/alacritty/alacritty) - Terminal emulator
- [Thunar](https://gitlab.xfce.org/xfce/thunar) - File manager
- [Zathura](https://git.pwmt.org/pwmt/zathura) - Document viewer
- [Firefox](https://www.firefox.com/) - Browser
- [Thunderbird](https://www.thunderbird.net/) - Mail client

For more details on the software I'm running feel free to check out my [dotfiles](https://github.com/piero-vic/dotfiles)
repo.
